# Road Accident Analysis Dashboard (Excel)

## Overview

I built this dashboard to practice Excel dashboard development using a real-world road accident dataset. The objective was to convert raw data into a simple, interactive report that highlights accident trends and key performance indicators.

The dashboard allows users to explore accident data using different filters and quickly understand how factors such as road type, weather conditions, vehicle type, and accident severity affect the overall number of casualties.

## Dashboard Includes

- Casualties by Vehicle Type
- Monthly Casualty Trend (Current Year vs Previous Year)
- Casualties by Road Type
- Casualties by Road Surface
- Area-wise Casualties (Urban vs Rural)
- Casualties by Light Condition
- Interactive Filters for:
  - Year
  - Weather Condition
  - Wind Condition
